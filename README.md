# ghgrab

Download specific files and folders from GitHub repositories without cloning.

[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)
[![Rust](https://img.shields.io/badge/built_with-Rust-orange.svg)](https://www.rust-lang.org/)

## Features

- Terminal UI for browsing repos
- Download individual files or folders
- Git LFS support
- Direct URL argument support

## Install

```bash
npm install -g ghgrab
```

## Usage

```bash
ghgrab
# or
ghgrab https://github.com/owner/repo
```

## Keys

| Key | Action |
| --- | --- |
| ↑↓ | Navigate |
| Enter | Open |
| Space | Select |
| a/u | All/None |
| d | Download |
| q | Quit |
